# Full Throttle Revenue Presentation

Interactive F1-themed RevOps presentation for Manning School of Business at University of Massachusetts Lowell

## Project Overview
This project is an interactive, F1 racing-themed presentation on Revenue Operations (RevOps) and Automation designed for undergraduate business students. It features dynamic slide transitions, interactive polling, and animated timelines.

## Features
- F1 racing-inspired design
- Interactive real-time polls using Supabase
- Responsive design for all devices
- Admin dashboard for presentation control

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Express.js
- Database: Supabase
- Hosting: Vercel
